Nalla-Nudi
Bilingual Technical Learning App for Kannada Medium Students

Nalla-Nudi is an Android educational application developed using Kotlin and Jetpack Compose to help Kannada-medium students understand technical English terminology through simplified Kannada explanations, pronunciation support, and offline learning features.

The application is designed to reduce the language barrier faced by students transitioning into English-based technical education, especially in subjects such as Computer Science, Science, Mathematics, and Engineering.

Features
Technical word search
Subject-wise category filtering
English pronunciation support
Kannada pronunciation support
Offline glossary access
“My List” revision feature
Word of the Day
Modern UI using Jetpack Compose
Responsive Android application

Application Workflow
User opens the application
Searches for a technical word
Views Kannada explanation and details
Listens to pronunciation
Saves important words to “My List”
Revises saved words offline anytime
Problem Statement

Kannada-medium students often struggle to understand technical English vocabulary during higher education due to language barriers and lack of simplified subject-specific learning resources.

Objective

The objective of Nalla-Nudi is to provide an accessible bilingual educational platform that helps students learn technical terminology more effectively through localized explanations and pronunciation assistance.
